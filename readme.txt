This application is for storing customer database.

Feutures: CRUD customer 

Endpoints:    http://localhost:8080/crm.remade/list       ===> List of customer
              http://localhost:8080/crm.remade/add        ===> Add new customer
              http://localhost:8080/crm.remade/save       ===> Save new/update customer
              http://localhost:8080/crm.remade/update     ===> Update customer data
              http://localhost:8080/crm.remade/delete     ===> Delete customer

Environment: 
            1. Spring Framework 5.2.5.RELEASE
            2. Java 8
            3. Hibernate
            4. MySQL Server 5.5
            5. Tomcat 9
